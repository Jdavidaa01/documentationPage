    JavaScript (JS) es un lenguaje de programación que se utiliza para crear páginas web interactivas y dinámicas. Se ejecuta en el navegador web y permite: Actualizar fuentes de redes sociales, Mostrar animaciones y mapas interactivos, Controlar multimedia, Validar formularios, Actualizar contenido en tiempo real.
    JS es la tercera capa de las tecnologías web estándar, junto con HTML y CSS. Se ejecuta directamente en el navegador del usuario, por lo que no es necesario compilarlo previamente.

    Algunas caracteristicas de JS son:

    Simplicidad, Velocidad, Versatilidad, Popularidad, Carga del servidor, Actualizaciones.
    JS se puede utilizar también para el desarrollo de aplicaciones móviles, gracias a frameworks y tecnologías como React Native y Progressive Web Apps (PWA).
    Algunos de los principales frameworks de JavaScript son: Ember, Singular, React, Vue.js.

    Funciones en Js

    Una función de JavaScript se define con la palabra clave function, seguida de un nombre, seguido de paréntesis ().
    Los nombres de las funciones pueden contener letras, dígitos, guiones bajos y signos de dólar (las mismas reglas que las variables).

    Los paréntesis pueden incluir nombres de parámetros separados por comas: (parámetro1, parámetro2, ...)
    El código a ejecutar, por la función, se coloca entre llaves: {}

    Se puede llamar a funciones de dos maneras

    • Funciones generales
    • Funcion de flecha
    function myFunction(a, b) {
    // Function returns the product of a and b
    return a * b;
    } //Funcion forma de flecha let func = (arg1, arg2, ..., argN) => expression;
    Objetos y metodos

    En JavaScript, un objeto es una colección de propiedades y métodos, mientras que un método es una función que se define dentro de un objeto:

    Objeto
    Es un grupo de pares clave-valor que puede contener propiedades, métodos, cadenas, números, booleanos, y otros tipos de datos. Los objetos se pueden comparar con los objetos de la vida real

    Método
    Es una función que se define dentro de un objeto y que se usa para manipular los datos almacenados en él. Los métodos se crean usando funciones y se ejecutan cuando se les llama con los paréntesis ().
    Los métodos se utilizan para definir las operaciones que se pueden realizar con un objeto. Por ejemplo, el método length se utiliza para averiguar la longitud de una cadena.
    Para acceder a las propiedades y métodos de un objeto en JavaScript, se puede usar la notación de puntos o la notación de corchetes. La notación de puntos es la más utilizada, pero tiene algunas limitaciones.

    const car = {type:"Fiat", model:"500", color:"white"};
    Java Exceptions Try&Catch
    Java Exceptions

    Al ejecutar código Java pueden ocurrir diferentes errores: errores de codificación cometidos por el programador, errores por entradas incorrectas u otras cosas imprevisibles.
    Cuando ocurre un error, Java normalmente se detendrá y generará un mensaje de error. El término técnico para esto es: Java generará una excepción (arrojará un error).

    Java try and catch

    La declaración try le permite definir un bloque de código que se probará en busca de errores mientras se ejecuta.
    La declaración catch le permite definir un bloque de código que se ejecutará si ocurre un error en el bloque try.
    Las palabras clave try and catch vienen en pares:

    try { tryStatements } catch (exceptionVar) { catchStatements } finally { finallyStatements }
    Web API
    ¿Qué es la API web?

    API stands for Application Programming Interface.
    A Web API is an application programming interface for the Web.
    A Browser API can extend the functionality of a web browser.
    A Server API can extend the functionality of a web server.

    API de navegadores

    Todos los navegadores tienen un conjunto de API web integradas para admitir operaciones complejas y facilitar el acceso a los datos.
    Por ejemplo, la API de geolocalización puede devolver las coordenadas de la ubicación del navegador.

    fetch('https://jsonplaceholder.typicode.com/users', { method: 'POST', // Establecemos el método de la petición aqui }) .then(respuesta => respuesta.json()) .then(datos => console.log(datos))
